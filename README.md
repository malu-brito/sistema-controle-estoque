# 📦 Sistema de Controle de Estoque

Aplicação web desenvolvida para gerenciamento de produtos em estoque, permitindo operações básicas de cadastro, listagem e remoção de itens.

## 🚀 Funcionalidades

- Cadastro de produtos
- Remoção de produtos
- Listagem de itens em estoque

## 🛠️ Tecnologias utilizadas

- JavaScript
- HTML
- CSS

## 🎯 Objetivo

Projeto desenvolvido com foco no aprendizado de lógica de programação e manipulação de dados em JavaScript.

## 📚 Aprendizados

- Estruturas condicionais (if/else)
- Estruturas de repetição (for/while)
- Manipulação de arrays
- Organização de código

## 💡 Melhorias futuras

- Interface mais interativa
- Persistência de dados (banco de dados)
- Edição de produtos
